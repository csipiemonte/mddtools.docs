Diagramma complessivo

Data la complessità del metamodello non è possibile allegare una immagine della struttura delle classi/associazioni. E'possibile visualizzare in Eclipse il metamodello scaricando i seguenti file ed aprendo il diagramma.

  • Metamodello guigen: guigen.ecore
  • Diagramma del metamodello guigen: guigen.ecorediag
