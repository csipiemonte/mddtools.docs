Class "datagen :: CustomUpdater"

Defined in package : datagen

Operazione di aggiornamento dati altamente personalizzabile, tramite scrittura di codice custom. E' utile, ad esempio, nel caso si debbano realizzare delle operazioni di aggiornamento di un insieme di righe e non di una singola riga identificata da PK.

Attribute ecore :: EString filterClassFQN

Full qualified name della classe che deve essere utilizzata come primo parametro di input per il finder. Serve per determinare l'inisieme di righe su cui operare la modifica Possibili valori:

  • Se non si specifica la proiprietà la classe utilizzata sarà java.lang.Object.
  • Se il valore specificato è $dto, sarà utilizzata la classe del dto associato.
  • Se il valore specificato è $dto[], sarà utilizzata una ArrayList costituita da oggetti del tipo del dto associato.
