Descrizione

E' il classico widget utilizzabile per la selezione di un file che dovrà essere inviato al server (funzione di file upload). Il FileUpload può essere utilizzato all'interno di:
  • WidgetsPanel, con layout Verticale, Orizzontale, Griglia, nel caso sia necessario associare il pulsante ad altri widget (es. pulsanti di lookup)
Prevede due modalità di funzionamento: standard e ricca.

modalità standard

Il funzionamento di base del FileUpload prevede:
  • la possibilità di selezionare da file system un file mediante browsing sulle cartelle locali alla postazione client
  • la possibilità di essere disabilitato a comando
  • la possibilità di essere reso invisibile a comando
  • la possibilità di essere disabilitato/invisibile a fronte di regole associate al profilo dell'utente collegato

modalità ricca

La modalità ricca prevede di default i seguenti comportamenti aggiuntivi:
  • tooltip al passaggio del cursore sulla label del widget

Eventi associati al widget

evento "event-clicked"

Il widget FileUpload contiene un pulsante che può ricevere un evento di pressione da parte dell'utente a fronte del quale viene visualizzato il dialog di selezione del file da sottomettere al server. Questo evento è implicito (non occorre definire un EventHandler per attivarlo). A fronte di questo evento non è però possibile eseguire logica di business.
