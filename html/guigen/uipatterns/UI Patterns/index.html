Sezione UI Patterns (v.1.2.0)

In questa sezione sono descritti tutti gli elementi di base disponibili per la realizzazione della user interface di una applicazione gestionale.

Tali elementi sono stati descritti all'interno di un catalogo articolato in tre sezioni:

  • Widgets
  • Panels
  • Layout

La definizione del catalogo ha come obiettivo la condivisione trasversale, a tutte le discipline di ingegneria del software (analisi, progettazione, sviluppo e testing), dei concetti e dei contenuti relativi alla gestione delle user interface. Con questo prodotto si intende avviare un processo di omogenizzazione del linguaggio di comunicazione che permetta così di ridurre il gap informativo che spesso si manifesta nel ciclo di realizzazione del software.

Tale processo si concretizza inizialmente con la definizione di un primo set di widgets e si estenderà successivamente all'approfondimento delle sezioni relative a panels e layout. I widgets definiti rappresentano un primo insieme significativo, probabilmente non esaustivo, di oggetti grafici usati negli applicativi gestionali prodotti dal CSI Piemonte.

Per ogni widgets sono illustrate le caratteristiche relativamente agli aspetti di grafica, dinamica di interazione, struttura e dati. Per ognuna di esse si cercherà di fornire un quadro di utilizzo nei diversi ambienti implementativi; in prima istanza tale mappatura è disponibile nell'ambito degli strumenti MDD.
