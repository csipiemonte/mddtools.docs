Elemento di catalogo: ComboBox ()

Descrizione

E' il widget di selezione di uno o più elementi a partire da una lista di dati dinamicamente precaricata. La collezione dei dati che compongono l'insieme selezionabile solitamente viene desunta da una struttura dati, i cui elementi devono essere specificatamente indicati.

Quando il controllo non è attivo mostra solo un valore (corrispondente al contenuto memorizzato nel campo all'ultima scelta effettuata); una volta attivato mostra tutta la lista dei possibili valori.

Il widget completo prevede una label ed il campo di selezione associato.

La label può essere, in casi particolari, vuota.

Alla selezione di un elemento dalla lista viene eseguita la logica applicativa associata.
