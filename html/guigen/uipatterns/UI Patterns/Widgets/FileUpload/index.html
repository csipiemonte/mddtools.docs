Elemento di catalogo: FileUpload ()

Descrizione

E' il classico widget utilizzabile per la selezione di un file che dovrà essere memorizzato in modo persistente nel sistema (funzione di file upload).

La selezione del file avviene tramite browsing sulle cartelle accessibili dalla postazione. La verifica del buon esito dell'operazione di upload del file deve essere gestita a livello applicativo.
