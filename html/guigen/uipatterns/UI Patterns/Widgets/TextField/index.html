Elemento di catalogo: TextField ()

Descrizione

E' il classico widget di immissione di dati testuali a riga singola. Un textfield può contenere un campo di tipo stringa, numero, data. Il widget completo prevede una label ed il campo di immissione associato. La label può essere, in casi particolari, vuota.
