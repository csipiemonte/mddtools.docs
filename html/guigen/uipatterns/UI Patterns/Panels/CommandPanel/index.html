Elemento di catalogo: CommandPanel ()

Descrizione

Il CommandPanel è un pannello utilizzato per realizzare pulsantiere, ovvero gruppi di Button destinati ad eseguire funzioni di business.
All'interno del singolo tab è possibile inserire tutte le tipologie di pulsanti disponibili.

Caratteristiche (features)

cod.nomedescrizione
struttura
PCMD001 contenimento widget

Può contenere i seguenti widget:

  • Button
  • ResetButton

PWDG003 layout supportati

I widget contenuti possono essere organizzati secondo i seguenti layout:

  • HorizontalLayout
  • UDLRCLayout
