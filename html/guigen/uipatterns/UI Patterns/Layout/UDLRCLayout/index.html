Elemento di catalogo: UDLRCLayout ()

Descrizione

L' UDLRCPanelLayout (layout a cinque quadranti Up, Down, Left, Right, Center) prevede che gli elementi contenuti nel pannello che possiede questo layout siano visualizzati in uno dei quadranti. realizzando in questo modo stutture di posizionamento relativo molto flessibili. Tipicamente questo layout è utilizzato nella disposizione di sotto-pannelli. Il layout può anche essere utilizzato per la disposizione di pulsanti all'interno di un CommandPanel.

Caratteristiche (features)

cod.nomedescrizione
struttura
LBRD001 disposizione a 5 quadranti Gli elementi contenuti nel pannello a cui è associato questo layout sono disposti in 5 quadranti (vedi figura):
  • UP: quadrante superiore
  • DOWN: quadrante inferiore
  • LEFT: quadrante sinistro
  • RIGHT: quadrante destro
  • CENTER: quadrante centrale
