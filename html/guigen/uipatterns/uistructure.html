UI Structure - pannelli utilizzabili per la struttura della U.I

In questa sezione sono descritti i principali pannelli con cui è possibile strutturare la User Interface con GUIGEN.

Pannelli e layout

La struttura della user interface in GUIGEN è corrisponde al modo in cui sono organizzate le varie porzioni di ciascuna schermata. La user interface si struttura in:

  • pannelli, che possono contenere a loro volta:
    • altri pannelli (sottopannelli)
    • widget (campi, pulsanti)
La collocazione spaziale dei sottopannelli all'interno dei pannelli e dei widget all'interno dei pannelli è determinata, oltre che dalla gerarchia di inclusione dei vari elementi, anche dalle caratteristiche di Layout, che sono determinate da:
  • il layout posseduto dal singolo pannello, che determina la modalità con cui i suoi sotto elementi (sottpannelli o widget) possono essere organizzati
  • una specificazione di layout posseduta dal singolo sotto-elemento (sottopannello, widget) che determina quale "posizione" occuperà quell'elemento all'interno del layout del suo contenitore. (per alcuni tipi di layout tale specificazione è implicita - es. nel Layout Verticale la specifica della posizione di un elemento è semplicemente data dall'ordine in cui appaiono i vari sotto-elementi nel modello)
