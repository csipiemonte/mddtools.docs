description

Lo UserInfoPanel è un pannello utilizzato per visualizzare le informazioni relative all'utente selezionato, che tipicamente sono:

  1. il nome ed il cognome dell'utente
  2. il codice fiscale dell'utente
  3. l'ente di appartenenza
  4. il ruolo
