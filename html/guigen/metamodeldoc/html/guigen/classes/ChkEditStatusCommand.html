Class "guigen :: ChkEditStatusCommand"

Defined in package : guigen

Permette di verificare se, all'interno di una sessione di editing, vi sono state delle variazioni al valore di uno o più ApplicationData. Affinchè il check sia veritiero è necessario che gli ApplicationData oggetto di verifica siano un sottoinsieme dell'insieme di ApplicationData indicati nel comando BeginEditCommand che ha originato la sessione di editing (altrimenti non sarà disponibile il valore originario con il quale effettuare il check di modifica). A fronte dell'esito del check sarà possibile proseguire l'esecuzione con uno tra due comandi (referenziati nei due elementi di tipo CommandOutcome). Mediante l'attributo checkAggregationE'possibile specificare la logica di aggregazione del check relativamente ai vari ApplicationData referenziati:

  • OR: l'esito sarà di "occorsa variazione" se il valore di almeno uno dei dati referenziati avrà subito una modifica rispatto allo stato originale.
  • AND: l'esito sarà di "occorsa variazione" se il valore di tutti i dati referenziati avrà subito una modifica rispatto allo stato originale.
  • XOR: l'esito sarà di "occorsa variazione" se il valore di uno soltanto tra i dati referenziati avrà subito una modifica rispatto allo stato originale.
