Class "guigen :: MultiDataWidget"

Defined in package : guigen

Classe astratta dalla quale discendono tutte le classi che rappresentano widget associabili a collezioni. Un MultiDataWidget ha un concetto di collezione del widget che rappresenta l'insieme finito di possibili valori selezionabili.

Reference guigen :: AppDataBinding multiDataBinding

Il binding all'ApplicationData nel quale sono contenuti gli elementi dell'insieme di possibili valori. Il tipo dell'oggetto puntato deve essere un TypedArray e il componentType di questo TypedArray deve essere un ComplexType, poichè è necessario poter specificare il valore selezionato e la label descrittiva di tale valore (perciò servono almeno due field).
