Class "guigen :: RefreshViewCommand"

Defined in package : guigen

Permette di effettuare il re-rendering di una porzione di ContentPanel. Funziona solo se sono abilitate le fatures di interazione "ricca", nell'elemento TargetPlatform. Al fine di garnatire l'accessibilità, Nel caso della cartuccia "struts2", se il browser non supporta javascript, la feature effettua un "downgrade" automatico mantenendo il funzionamento seppur perdendo la funzione ricca (ovvero si ha il refresh completo della pagina). Essendo sottoclasse di CommandOnWidgets e CommandOnPanels è possibile specificare la porzione di ContentPanel che deve essere aggiornata. L'interpretazione delle proeprty targetWidgets e targetPanels dipende dalla particolare cartuccia di generazione utilizzata.
