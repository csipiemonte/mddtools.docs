Attribute ecore :: EBoolean editable

Se impostato a true la colonna è editabile. Il widget utilizzato per l'editing della colonna dipende dal tipo del field associato alla colonna (tramite la property selector):

  • CheckBox in caso di tipo boolean
  • TextField in tutti gli altri casi (stringhe, numerici, date)
Inoltre, se sono impostate le property multiDataBinding o multiDataPropertySelector, che comportano la possibilità di scegliere il valore da una lista di valori, sarà utilizzata una ComboBox

Attribute ecore :: EString multidataPropertySelector

Permette di associare ad una colonna editabile la possibilità di scegliere il valore della cella a partire da una collezione di coppie chiave/valore. Per specificare quale campo del record di cui è costituita questa collezione ausiliaria utilizzare come chiave e come descrizione, occorre impostare le property multidataKeySelector e multidataValueSelector. L'insieme dei valori è differenziato riga per riga . Nel caso sia sufficiente uno stesso insieme di valori per tutte le righe utilizzare multiDataBinding
