Class "guigen :: ComboBox"

Defined in package : guigen

MultiDataWidget che permette la selezione di uno o più valori all'interno di un elenco di possibili opzioni. Se il value-binding è impostato su un ApplicationData di tipo Simple la combo sarà a selezione singola; se il value-binding è impostato su un ApplicationData di tipo TypedArray (con component type Simple) la combo sarà a selezione multipla. Alla combo è possibile associare un event handler relativo all'evento VALUE_CHANGED, che viene scatenato nel momento un cui viene modificata la selezione.
