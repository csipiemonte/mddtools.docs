Class "guigen :: TabSetPanel"

Defined in package : guigen

Tipo specifico di MultiPanel corredato di componente di controllo a "tab": per passare da un tab all'altro è sufficiente selezionare la "linguetta" corrispondente. Come per tutti i MultiPanel il programmatore avrà nel model le informazioni riguardanti:

  • il tab correntemente visualizzato
  • il tab "target" (quello sulla cui 2linguetta" si è cliccato; questa informazione è disponibile solo all'interno di un EventHandler associato al widget fittizio TabSwitcher)
    • .
Il primo tab è quello visualizzato di default.

description

Il TabSetPanel è un pannello utilizzato realizzare strutture organizzate a tab. Prevede due sezioni visibili:

  1. la lista di " linguette " che servono per attivare i vari tab (con evidenza visiva del tab correntemente selezionato)
  2. la sezione in cui viene visualizzato il contenuto del tab correntemente selezionato.
All'interno del singolo tab è possibile inserire tutte le tipologie di pannello disponibili.
