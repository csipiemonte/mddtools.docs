Class "guigen :: Breadcrumb"

Defined in package : guigen

Rappresenta il cosiddetto "filo d'arianna", utilizzabile per mostrare all'utnete collegato il contesto applicativo in cui si trova (es. funzionalità corrente, livello di dettaglio, ...). E' un MultiDataWidget collegabile ad ApplicationData di tipo array di stringhe che è utilizzato per mantenere l'elenco degli step del filo d'arianna. La costruzione degli step deve essere effettuata tramite logica di business.
