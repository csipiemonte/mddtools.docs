Class "guigen :: GridPanelLayout"

Defined in package : guigen

Layout a griglia. I widget vengono inseriti in una griglia delle dimensioni specificate da columns e rows. Ogni cella contiene un widget e la corrispondente label, se specificata. Per specificare la posizione nella griglia di un widget occorre utilizzare il GridWidgetLayoutSpec per ogni widget contenuto nel pannello che ha questo layout.

description

Il GridPanelLayout (layout a grigia) è applicabile esclusivamente al pannello WidgetsPanel e prevede che gli elementi contenuti nel pannello che possiede questo layout (widget) siano visualizzati in una struttura a griglia. E' necessario specificare le dimensioni orizzontali e verticali della griglia e ciascuna cella della griglia può contenere al massimo un solo Widget. In aggiunta un widget può estendersi orizzontalmente per più di una cella (hspan), Di fatto, se il numero di colonne previste dal layout è n, occorre tenere in considerazione una struttura costituita da n colonne logiche e 2*n colonne fisiche (una per la label ed una per il campo effettivo, il tutto moltiplicato per il numero colonne logiche). E' possibile regolare le percentuali di occupazione della componente label e della componente widget di ciascuna delle colonne logiche. Le percentuali sono relative allo spazio a disposizione dell'interno pannello. Nel clacolo occorre inoltre tenere in considerazione anche casi " degeneri " quali:

  • label non valorizzate
  • span orizzontali maggiori di 1
