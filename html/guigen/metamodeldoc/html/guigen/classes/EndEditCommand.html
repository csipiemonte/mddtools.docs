Class "guigen :: EndEditCommand"

Defined in package : guigen

Permette di terminare una sessione di editing su un certo insieme di ApplicationData, con la possibilit√† di effettuare un undo sui dati specificati. Utilizzato in congiunzione con BeginEditCommand. Nota: data la struttura dei due comandi di inizio e fine della sessione di editing è possibile effettuare l'undo di una parte degli application data specificati nel comando di inizio editing (specificando nel EndEditCommand solo una parte degli application data precedentemente specificati nel BeginEditCommand. Si consiglia, in questi casi, di terminare correttamente la sessione di editing per tutti gli application data per evitar eche rimangano in memoria dati di backup obsoleti. Per far ciò è sufficiente utilizzare due volte il comando EndEditCommand: la prima volta con undo impostato a true (e specificando solo i dati di cui si vuole effettuare l'undo) e una seconda volta con undo impostato a false, specificando i restanti dati.
