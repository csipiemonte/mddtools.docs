Class "guigen :: DataWidget"

Defined in package : guigen

Classe astratta dalla quale discendono tutte le classi che rappresentano widget associabili a dati (ApplicationData). Un DataWidget ha un concetto di valore del widget che rappresenta l'input inserito.
