Attribute guigen :: SimpleTypeCodes code

Codice che serve per discriminare il tipo base (scalare) specifico. Se è impostato userDefined a false allora il tipo corrisponde ad un tipo base del linguaggio host (es int, java.lang.Double). Se invece userDefined è impostato a true, viene generato un nuovo tipo di dato astratto basato sul tipo scalare corrispondente e nel quale è possibile specificare meglio il dominio di valori ammessi e alcune caratteristiche di formattazione.
