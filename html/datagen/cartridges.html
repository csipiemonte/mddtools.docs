Datagen - Cartridges disponibili

In questo capitolo sono descritte le cartucce (cartridge) di generazione disponibili.

cartuccia di generazione dello strato DAO

La generazione dello strato DAO si realizza mediante la cartuccia it/csi/mddtools/datagen/workflow/basicDao.mwe, che prevede i seguenti parametri:

parametrodescrizioneesempioobbl/opz
model percorso del file del modello datagen, a partire dalla root del workspace eclipse myprod.mdd/src/model/myprod/mycomp/mydao.datagen obbl
daoTargetProjectName nome del progetto java in cui devono essere generati i sorgenti mycomp obbl
targetDbms dbms per cui devono essere generati i dao. Può valere:
  • oracle
  • postgresql
 oracle obbl
basePackage package java a partire dal quale generare le classi impostando it.csi.prod.comp il package dei dao sarà generato a partire dal package it.csi.prod.comp.business.dao obbl
daoBeansFileName posizione del file dao-beans-xml percorso relativo alla root del progetto generato dove verrà creato il file dao.beans.xml. N.B: nel caso di utilizzo di datagen all'interno di un progetto master che già crea il file, l'ordine di richiamo delle due cartucce è rilevante: occorre richiamare prima la cartuccia del progetto master e poi quella del progetto datagen obbl

Esempio: 

    
<cartridge file="it/csi/mddtools/datagen/workflow/basicDao.mwe"
  model="myprod.mdd/src/model/myprod/mycomp/mydao.datagen" 
  daoTargetProjectName="mycomp"
  basePackage="it.csi.prod.comp.business"     
/>
